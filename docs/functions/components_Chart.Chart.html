      Function

      Komponent wyświetlający wykres słupkowy liczby logowań użytkownika w podziale na dni tygodnia. Wykorzystuje bibliotekę Recharts do renderowania wykresu oraz komponenty UI z @/components/ui/chart.

      Chart

      Parameters

      • props: { chartData: ChartData[]; hasFetched: boolean; user: null | User }

        Właściwości komponentu.

        • chartData: ChartData[]

          Dane do wykresu, zawierające informacje o logowaniach w poszczególnych dniach tygodnia.

        • hasFetched: boolean

          Flaga wskazująca, czy dane zostały pobrane.

        • user: null | User

          Obiekt użytkownika, określający, czy użytkownik jest zalogowany.

      Returns Element

      Wykres słupkowy z liczbą logowań lub komunikat o stanie (np. "Zaloguj się", "Ładowanie...").

      Example

      import { Chart } from '@/components/Chart';

      const user = { first_name: "patryk", id: "67eb5b5d-4b18-482c-a338-f761e5086811", last_name: "zawadzki", login: "user123@gmail.com" };
      const chartData = [
        { month: "Poniedziałek", logins: 5 },
        { month: "Wtorek", logins: 10 },
      ];
      const hasFetched = true;

      <Chart user={user} chartData={chartData} hasFetched={hasFetched} />

      Remarks

      • Komponent wymaga, aby user był obiektem (lub null), aby określić, czy wyświetlić wykres czy komunikat o logowaniu.
      • chartData powinno być zgodne z interfejsem ChartData (np. { month: string, logins: number }).
      • Jeśli hasFetched jest false, wyświetlany jest komunikat "Ładowanie...".
